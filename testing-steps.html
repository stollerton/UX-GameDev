Testing
Every feature and mechanic in the game needs to be tested for quality control. A game that hasn’t been thoroughly tested is a game that’s not even ready for an Alpha release. Is the character dialogue stale and boring? Are there features I can use to exploit the game? Questions like these need to be asked and answered.
UX Methods for the Testing Stage
0 out of 3
A/B Testing
A/B testing is offering alternative versions of a product to different users and comparing the results in order to find out which one performs better. This is a great technique for optimizing funnels and landing pages.

Eye Movement Tracking
Using technology to analyze the user’s eye movements across the interface. This provides data about what keeps users interested and what attracts their attention on the screen.

Usability Testing
Usability testing is the observation of users trying to carry out tasks with a product. Testing can be focused on a single process or be much more wide ranging.
